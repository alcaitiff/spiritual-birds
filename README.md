# spiritual-birds

A game made with Phaser 3 and Vuejs.

## How to play

Arrows to flap around
"q" to shoot
"w" to use a powerup
"p" to pause

## Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

### Compiles and minifies for production

```bash
yarn build
```

### Lints and fixes files
